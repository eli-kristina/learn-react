## Setup in Local


``` bash
# clone from repository
$ git clone https://github.com/eli-kristina/learn-react.git

# go to the working directory
$ cd learn-react

# rename .env.exampe file to .env
# open .env file and edit REACT_APP_SERVER_URL value

# install dependencies
$ npm install

# serve with reload at localhost:3000
$ npm run start
```
